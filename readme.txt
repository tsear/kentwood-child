== Description ==
- Inherits all core features from the Kentwood parent theme
- Custom SCSS architecture with mobile-first responsive design
- Gutenberg-friendly layout overrides
- Enhanced performance styles for image handling and typography
- Sticky navigation and custom button components
- Integrated stock ticker and RSS feed components
- Fully modular and production-ready
